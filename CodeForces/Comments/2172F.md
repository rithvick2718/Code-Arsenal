Given, it's clear that we need to make a minimum spanning tree to satisfy the needs of the given problem.

Now suppose $0< u < n-1$, and let $v \ne u$ and $v \in \{ 0, 1, \cdots n-1 \}$. If $ u > v $ then $\text{edge}(u,v) = \text{gcd}(p_v,p_{v+1}, \cdots, p_u) \ge \text{gcd}(p_0,p_1, \cdots p_u) $ similarly if $ u < v $ then $\text{edge}(u,v) = \text{gcd}(p_v,p_{v+1}, \cdots, p_u) \ge \text{gcd}(p_u,p_{u+1}, \cdots p_{n-1}) $ so we have any
$$
\text{edge}(u) \ge \text{min}(\text{gcd}(p_0,p_1, \cdots p_u),\text{gcd}(p_u,p_{u+1}, \cdots p_{n-1}))
$$
And we know equality holds.

So for nodes $u$ : $0< u < n-1$ we choose the best edges, and that means we have chosen $n-2$ edges and each $u$ is connected to $0$ or $n-1$ (based on which edge is smaller). So we have $n-2$ edges and $2$ components.

Now we can also say that W.L.O.G. for all $u<v$ $\text{edge} (u,v) = \text{gcd}(p_u,p_{u+1}, \cdots, p_v) \ge \text{gcd}(p_0, p_1, \cdots, p_{n-1}) = \text{edge}(0,n-1)$, Thus $\text{edge}(0,n-1)$ is the smallest edge in the graph and adding this to our collection completes our MST.

This is an MST as : it has $n-1$ edges, it is connected and each edge using edge analysis we can see that each edge has the lowest weight possible.
